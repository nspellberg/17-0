# 17-0

17-0 is a React/Next.js NFL all-time offensive draft game. Spin for a random team and decade, draft one player from that era's offense, watch the live lineup fill in, and reveal a predicted 17-game season record.

## Run

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Included

- Landing, how-to-play, draft with live lineup, rating reveal, final results, and design-system screens.
- Local all-time NFL offensive roster data across teams and decades.
- Mobile-first responsive UI with desktop war-room layouts.
- Scoring model based on offense, star power, lineup balance, chemistry, and completion.
